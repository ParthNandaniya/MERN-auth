# MERN-auth

Front end of MERN-server project

### Make sure MERN-server is up and running first

### to clone:
1. ``` git clone https://github.com/ParthNandaniya/MERN-auth.git ```
1. ``` cd MERN-auth ```
1. ``` yarn install ```
1. ``` npm start ```

1. go to ** http://localhost:3000/ ** on your browser

Happy coading...
